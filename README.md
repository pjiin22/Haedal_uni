## ►프로젝트 개요

강의실을 이용해 공부하고 싶지만, 실제로 자리를 찾아다니다 보면 겪는 불편들이 있습니다.

- 도서관 회의실은 1인 사용 불가
- 외부 스터디룸은 비용 부담이 큼
- 빈 강의실 여부는 일일이 찾아가 확인해야 함
- 특히 시험 기간엔, 노트북을 펼칠 조용한 공간조차 구하기 힘들다!

이러한 문제를 해결하기 위해,

경북대학교 **IT대학 소속 학우들**을 위한 **빈 강의실 관리 시스템**을 개발했습니다.

본 서비스는 **강의실 정보 + 사용자 신뢰도 + AI 예측**을 기반으로

더욱 정확하고 편리하게 학습 공간을 추천합니다.

## ►주요 기능

### 1. 확률 기반 강의실 예측 📊

- 시간표 데이터를 기반으로 **해당 시간 강의실이 비어 있을 확률**을 계산합니다.
- 사용자의 신뢰도(온도)에 따라 예측 정확도를 조정합니다.
    
    **예측 로직 흐름 - 가중치 적용 예측 보정**
    
    → 각 사용자의 신뢰도에 따라 예측 확률에 가중치 적용
    
    - ✅ **신뢰 높은 사용자** 예약 → 실제 이용 확률 ↑ → **공실 확률 ↓**
    - ❌ **신뢰 낮은 사용자** 예약 → no-show 가능성 ↑ → **공실 확률 ↑**
    

### **2. 보증 포인트 기반 신뢰 예측 시스템 🛡️**

반복되는 **허위 예약**과 **퇴실 미처리** 문제를 방지하고, **예측 정확도를 높이기 위해** 사용자의 신뢰도를 수치화 한 **보증 포인트 제도**를 도입했습니다.

- **보증 포인트**는 강의실 이용 이력을 기반으로 부여되며,
    
    성실하게 입실·퇴실한 사용자는 점수가 올라가고,
    
    허위 예약이나 미처리 시에는 감점됩니다.
    
- 예측 시스템은 **보증 포인트가 높은 사용자의 예약일수록 빈 강의실 확률에 더 높은 가중치**를 반영합니다.
- 결과적으로, **신뢰할 수 있는 사용자 중심의 예측 보정이 이루어져 정확도가 향상**됩니다.

> ✈️ 항공 오버부킹 시스템처럼, 신뢰 기반의 예측 우선순위를 적용한 구조입니다.
> 

### 3. 사용자 인증 기반 상태 변경 📸

- 사용자가 강의실을 '사용 중' 으로 변경하거나 퇴실할 때 **사진을 업로드**하도록 요청합니다.
- 이미지 분석 AI 연동 → **부정 신청 방지 및 자동 판별 가능**

### 4. 부가 기능

- **이용 내역 조회**: 예약 / 사용 중 / 종료 내역 확인 가능
- **강의실 검색**: 원하는 강의실의 상태를 단번에 알아내기 가능
- **강의실 상태**: 건물별 빈 강의실, 이용 불가 강의실 표시

# 🎨 FE(Frontend)

## ►주요 기능

### 1️⃣ 강의실 현황 표시

- **건물별 강의실 현황 시각화**
    - 예: IT 1호관 / 5호관 별로 강의실 수, 사용 가능 상태 구분
    - 강의실 개수 및 상태별 분류(총 개수, 빈 강의실 개수 등)
- **공실 예측 확률 기반 분류**
    - ▸ **높은 확률(70% 이상)**
    - ▸ **중간 확률(40~70%)**
    - ▸ **낮은 확률(40% 미만)**
- **잠김 여부 표시**
    - 🔒 잠긴 강의실은 "입장 불가" 상태로 버튼 비활성화
    - 🔓 열린 강의실만 입장 가능
- **사용 여부 상태**
    - ▸ 사용 가능 / 사용 중 / 미사용 으로 구분
    - *잠김 상태는 항상 “미사용”으로 표시*

---

### 2️⃣ 이용 내역 관리

- **개인 이용 이력 확인**
    - 예약 시간, 사용 여부, 종료 여부가 정리된 리스트 제공
- **상태에 따른 사인 자동 변경**
    - 예약 중 → "현재 예약 중"
    - 사용 인증 후 → "현재 이용 중"
- **남은 시간 표시**
    - 사용 시작 이후에는 **실시간 남은 시간 계산**
    - 예약 상태일 경우에는 남은 시간 숨김 처리

---

### 3️⃣ UI/UX 디자인

- **Figma 기반 설계**
    - 실사용자 입장에서 이해하기 쉬운 화면 구성
    - 버튼/배지/색상 기준 통일
- **시각적 강조**
    - 공실 확률에 따른 색상 기반 상태 시각화
        - 초록색: 높은 공실 확률
        - 주황색: 중간 확률
        - 붉은색: 낮은 확률
    - 아이콘 활용 (🔒 잠김, 👤 사용 중,  ✅사용 가능 등)

## ►주요 기능

### 1️⃣ 사용자 인증 및 신뢰도 연동

- 학번, 이름, 전화번호를 입력 받아 **학생 인증 처리**
- 인증된 사용자에 대해 **세션 또는 토큰 기반 로그인 상태 유지**
- 사용자 신뢰도(보증 포인트)와 연동하여 **예측 정확도 향상**

---

### 2️⃣ 강의실 정보 관리

- 건물 및 강의실 목록 **DB에서 조회 및 제공**
- 시간표와 예약 정보를 기반으로 **시간대별 강의실 이용 가능 여부 판단**
- 강의실 상태 정보(잠김 여부, 사용 여부 등) API 제공

---

### 3️⃣ 강의실 예약 및 이용 기록 처리

- 예약 요청/취소 처리 → DB 반영 및 상태 갱신
- 예약 시간대 자동 추적 → 사용 중 상태 전환 및 시간 계산
- 사용자별 총 사용 시간, 이용 횟수 등 **이용 통계 API 제공**

---

### 4️⃣ 빈 강의실 예측 API

- KNU 시간표와 사용자 신뢰도를 바탕으로 **공실 확률 계산**
- 예측된 확률을 **퍼센트 값으로 API 응답** → 프론트엔드에서 시각화

---

### 5️⃣ 사진 기반 인증 시스템

- 사용자가 ‘사용 중’으로 상태 변경 시 **사진 업로드 요청**
- OCR 기반 AI 판별기로 사진 속 강의실 번호를 추출
- **예약한 강의실과 일치 여부 판별**하여 인증 처리

## ►확장 가능성 및 마무리

이 시스템은 단순한 강의실 예약에 그치지 않고,

**스터디카페, 독서실, 회의실 등 다양한 공간 관리 서비스**에 확장 적용이 가능합니다.

- 신뢰 기반의 예약 예측
- 실시간 공간 상태 시각화
- 사진 인증을 통한 실제 사용 검증

이러한 구조는 **공용 공간의 효율적 운영과 악용 방지**에 탁월한 효과를 보이며,

**대학 외에도 민간 공간 서비스 전반에 적용 가능한 유연한 시스템 구조**로 설계되었습니다.
